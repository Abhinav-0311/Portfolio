# Abhinav Jain Portfolio

This is a lightweight static portfolio site built for easy deployment on Vercel.

## Files

- `index.html` contains the structure and page sections.
- `styles.css` contains the full visual design and responsive layout.
- `script.js` contains the portfolio content arrays and reveal animations.
- `assets/Abhinav-Jain-CV.pdf` is the resume download linked from the page.

## Customize the content

The easiest way to personalize this site is to edit the data in `script.js`:

- Update `focusItems` for strengths or specializations.
- Update `projectItems` for real projects, links, and technologies.
- Update `timelineItems` for education, internships, and achievements.
- Replace the contact buttons in `index.html` with your real email, LinkedIn, or GitHub links.

You can also edit the headline and summary text in `index.html`.

## Deploy on Vercel

1. Push this folder to a GitHub repository.
2. Log in to [Vercel](https://vercel.com/).
3. Click **Add New...** then **Project**.
4. Import the GitHub repository.
5. Keep the default settings for a static site and deploy.

Because this is a plain HTML/CSS/JS site, no build command is required.
